# Journey 6: AI handoff

## Scenario
An engineer wants to use Claude/OpenClaw to automate a recurring reporting task.

## Trigger
A useful automation brief has been generated.

## Flow
1. User clicks “Copy as Agent Spec.”
2. User pastes into external AI tool.
3. AI tool produces:
   - a script
   - workflow plan
   - API integration outline
4. User implements or tests the result.
5. Friction Log asks one week later:
   “Did this reduce the friction?”
6. User marks:
   - Reduced

### User goal
Turn the brief into an implemented improvement.

### Product requirement
The handoff format must be clean and agent-compatible.

### Success criteria
- exported brief is usable without major rewriting
- user reports reduced or resolved friction
- app captures outcome
