# Journey 8: Privacy-conscious onboarding

## Scenario
A new user is interested in Friction Log but worried it is surveillance software.

## Trigger
First launch.

## Flow
1. App explains:
   - what is collected
   - what is not collected
   - local-first storage
   - no screenshots
   - no keystrokes
2. User is offered setup options:
   - app-only mode
   - app + title mode
   - excluded apps
   - pause shortcut
3. App suggests excluding:
   - password managers
   - banking apps
   - personal messaging apps
4. User enables app-only mode and excludes a few apps.
5. Tracking starts only after explicit confirmation.

### User goal
Feel safe before granting trust.

### Product requirement
Privacy must be legible and configurable from day one.

### Success criteria
- onboarding reduces creepiness
- user understands controls
- user can begin with a conservative setup
