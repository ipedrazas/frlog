
# Key Product Opportunity Themes

## 1. Make capture emotionally lighter than the friction itself
Best opportunities:
- global shortcut
- one-line input
- dot commands
- instant dismissal
- no required fields

## 2. Build trust through explainability
Best opportunities:
- show examples behind every cluster
- confidence scores
- editable tags
- correction loop
- visible privacy controls

## 3. Prioritize action over analytics
Best opportunities:
- top 3 weekly opportunities
- worth-fixing triage
- structured automation brief
- export workflows
- resolved/reduced tracking

## 4. Treat privacy as product, not policy
Best opportunities:
- app-name-only default
- title capture opt-in
- exclusion suggestions
- pause shortcut
- “what we collect / what we never collect” screen

## 5. Keep the tone compassionate
Best opportunities:
- “worth fixing” instead of “time wasted”
- “coordination-heavy” instead of “inefficient”
- “consistently deferred” instead of “failed to do”
- systems framing over personal blame
